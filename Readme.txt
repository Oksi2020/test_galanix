Перше та третє завдання написані без сторонніх бібліотек та запускаються за допомогою
відкриття в інтернеті сторінки index.html.
Друге завдання написано з використанням бібліотеки React.js (create react app). 
Для його запуску потрібно відкрити термінал в кореневій папці, та ввести команду npm init
(для того, щоб завантажити всі модулі залежні, оскільки node_modules знаходяться в 
gitignore). Після цього ввести команду npm start, яка відкриє додаток в браузері.
Перепрошую за невиконане бонусне завдання та недстатнє оформлення елементів до третьої 
частини. Трішки не вистачило часу.
